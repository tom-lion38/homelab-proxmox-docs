# Vue d’ensemble de l’architecture

Ce document décrit l’organisation générale de l’homelab Proxmox.

## Principes
- Séparation des rôles (services / supervision / tests)
- Simplicité et lisibilité
- Facilité de maintenance

## Éléments clés
- Proxmox comme couche centrale
- VM et CT selon les usages
- Réseau interne segmenté logiquement
